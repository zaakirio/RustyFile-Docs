## Why RustyFile?

<CardGrid stagger>
  <Card title="Blazing Fast" icon="rocket">
    Rust-powered backend with zero GC pauses. Sub-50ms startup, zero-copy file streaming, and ~15 MB memory footprint.
  </Card>
  <Card title="Single Binary" icon="seti:config">
    One binary, no runtime dependencies. The React frontend is embedded directly into the Rust binary via `rust-embed`. Just download and run.
  </Card>
  <Card title="Video Streaming" icon="laptop">
    Stream video with HTTP range requests and on-demand HLS transcoding via FFmpeg. Subtitles (VTT, SRT, ASS) are auto-detected.
  </Card>
  <Card title="Resumable Uploads" icon="up-caret">
    TUS 1.0 protocol support means large uploads survive network interruptions. Drag-and-drop with concurrent upload queues.
  </Card>
  <Card title="Secure by Default" icon="approve-check-circle">
    Path traversal prevention, JWT + Argon2id auth, rate-limited login, constant-time password checks, CSP headers, and atomic file writes.
  </Card>
  <Card title="Zero Config" icon="setting">
    Launch with a single command. First-run setup wizard creates your admin account. Configure via CLI flags, environment variables, or TOML.
  </Card>
</CardGrid>

## Quick Start

```bash
# Download and run
rustyfile --root /path/to/files

# Or with Docker
docker run -d -p 8080:8080 -v /your/files:/data rustyfile/rustyfile
```

Visit `http://localhost:8080` and create your admin account during the 5-minute setup window.
