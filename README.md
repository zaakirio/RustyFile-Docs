# RustyFile Docs

Documentation site for [RustyFile](https://github.com/zaakirio/rustyfile) — a fast, lightweight file server built with Rust.

**Live at [rustyfile.io](https://rustyfile.io)**

Built with [Astro](https://astro.build) + [Starlight](https://starlight.astro.build).

## Development

```bash
npm install
npm run dev       # Start dev server at localhost:4321
npm run build     # Build for production
npm run preview   # Preview production build
```

## Deployment

Automatically deployed to GitHub Pages on push to `master` via GitHub Actions.
